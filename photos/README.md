# Photo Documentation

## Organization Guide

Store all restoration photos in the appropriate subdirectory.

---

## Directory Structure

```
photos/
├── original/        # Initial condition documentation
├── disassembly/     # Taking apart process
├── restoration/     # Work in progress
└── completed/       # Finished restoration
```

---

## Photo Naming Convention

Use consistent naming for easy organization:

```
YYYY-MM-DD_category_description_##.jpg
```

**Examples:**
- `2026-01-30_engine_intake_manifold_01.jpg`
- `2026-02-15_napco_transfer_case_removal_03.jpg`
- `2026-03-20_body_cab_corner_repair_before_01.jpg`

---

## Best Practices

### Equipment
- Use good lighting (natural light or work lights)
- Keep camera/phone lens clean
- Take steady shots (use tripod if possible)
- Shoot at highest resolution available

### Technique
- Include reference objects for scale (ruler, coin)
- Photograph all sides/angles
- Document part numbers and stampings
- Take before AND after photos
- Shoot overall and detail shots

### Storage
- Backup photos regularly
- Keep originals at full resolution
- Consider cloud backup
