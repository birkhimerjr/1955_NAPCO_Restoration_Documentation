# 1955 Chevrolet Truck VIN Decoder

## Serial Number Identification Guide

---

## VIN Location

The 1955 Chevrolet truck serial number (VIN) is stamped on a **plate attached to the left door pillar** (hinge side of the driver's door).

---

## VIN Format

```
Example: H55J-001234

H   55   J   001234
│   │    │      │
│   │    │      └── Sequential Production Number
│   │    └── Assembly Plant Code
│   └── Model Year (1955)
└── Series Code
```

---

| Code | Series | Description |
|------|--------|-------------|

## Production Sequence Numbers

- **1955 Second Series** (Task Force) serial numbers began at 001001 for each plant
- Numbers are sequential by plant, not nationwide
- Lower numbers indicate earlier production

---

## Example Decodings

### Example 1: H55J-003456
```
H     = 3100 Series (1/2 Ton, 114" wheelbase)
55    = 1955 Model Year
J     = Janesville, Wisconsin assembly plant
003456 = Production sequence number
```

### Example 2: K55A-001892
```
K     = 3600 Series (3/4 Ton)
55    = 1955 Model Year
A     = Atlanta, Georgia assembly plant
001892 = Production sequence number
```

## Additional Identification

### Engine Number Location

**235 Thriftmaster Six:**
- Stamped on right side of block, towards rear
- Format varies

**265 Turbo-Fire V8:**
- Stamped on front of block, below right cylinder head
- Format varies

### Body Number Plate

Located below the VIN plate on the left door pillar:
- Style number
- Body number
- Trim combination
- Paint code

### Fisher Body Date Code

Stamped on body plate:
```
Example: 3C55

3  = Month (March)
C  = Week (third week)
55 = Year (1955)
```

## NAPCO Identification

### Transfer Case Tag
- Riveted aluminum plate on transfer case
- Contains NAPCO model and serial number

### Front Axle Markings
- Dana/Spicer casting numbers on differential housing
- NAPCO-specific components may have additional markings

### Documentation
- Original NAPCO trucks may have dealer installation records
- Check glovebox for any original paperwork

---

## Verification Tips

1. **Cross-reference all numbers** - Engine, body, and VIN should be consistent
2. **Check casting dates** - Components should predate assembly date
3. **Look for anomalies** - Replaced or re-stamped plates are red flags
4. **Document everything** - Photograph all identification plates and stamps
