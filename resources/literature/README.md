### Physical Copies
- Hemmings Motor News
- Swap meets (Hershey, Carlisle, etc.)
- Estate sales
- Automotive museum gift shops
- eBay and online auctions

### Digital Archives
- Old Car Brochures (oldcarbrochures.org)
- GM Heritage Center
- Internet Archive (archive.org)
- Enthusiast forums

### Reproduction Literature
- Jim Osborn Reproductions
- Auto Literature (autoliterature.com)
- Faxon Auto Literature

## File Naming Convention

```
[YEAR]_[TYPE]_[DESCRIPTION].[ext]

Examples:
1955_brochure_truck_lineup.pdf
1955_manual_shop_manual_section_6.pdf
1955_napco_installation_instructions.pdf
1955_ad_saturday_evening_post.jpg
```

---

## Scanning Guidelines

If scanning original documents:

| Setting | Recommendation |
|---------|----------------|
| Resolution | 300 DPI minimum, 600 for detailed |
| Format | PDF for documents, TIFF/PNG for photos |
| Color | Color scan, even for B&W originals |
| Backup | Store in multiple locations |
