## Manual Excerpts Organization

When adding manual excerpts, organize by system:

```
manuals/
├── engine/           # Engine service
├── transmission/     # Trans service
├── napco/            # NAPCO-specific
├── electrical/       # Wiring, charging
├── brakes/           # Brake service
├── suspension/       # Springs, steering
├── body/             # Body repair
└── specifications/   # Specs, torque values
```
