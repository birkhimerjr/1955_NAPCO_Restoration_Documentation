## VIN Decoding

### VIN Location

The serial number is stamped on a **plate attached to the left door hinge pillar** (driver side).

### 1955 First Series VIN Format

The 1955 First Series follows the same format used since 1947:

```
Example: 3100-12345 or variations including plant code

Format may include:
- Series designation (3100, 3600, 3800)
- Plant code letter
- Sequential production number
```

## Engine Specifications

### 235ci Thriftmaster Six (Standard & Only Option)

*Note: Unlike the 1955 Second Series, the First Series did NOT offer a V8 option.*

| Specification | Value |
|---------------|-------|
| **Type** | Inline 6-cylinder, OHV |
| **Displacement** | 235.5 cubic inches (3.86L) |
| **Bore** | 3.5625" |
| **Stroke** | 3.9375" |
| **Compression Ratio** | 7.5:1 (std), 7.1:1 (low compression) |
| **Horsepower** | 112 hp @ 3700 RPM |
| **Torque** | 200 lb-ft @ 2000 RPM |
| **Carburetor** | Rochester B, 1-barrel |
| **Fuel Requirement** | Regular leaded gasoline |
| **Oil Capacity** | 5 quarts (with filter) |
| **Cooling System** | 17 quarts (with heater) |

### Engine Tuning Specifications

| Specification | Value |
|---------------|-------|
| **Firing Order** | 1-5-3-6-2-4 |
| **Cylinder Numbering** | 1-6 front to rear |
| **Valve Clearance (hot)** | Intake: 0.006", Exhaust: 0.013" |
| **Spark Plugs** | AC 44 or equivalent |
| **Spark Plug Gap** | 0.035" |
| **Point Gap** | 0.016" |
| **Dwell Angle** | 35°-40° |
| **Ignition Timing** | 4° BTDC @ idle |

- **Engine Number Location:** Stamped on right side of block, near rear
- **Casting Number Location:** Left side of block, near oil filler

### NAPCO Option

NAPCO 4x4 conversion was a dealer-installed option:
- Not a factory RPO code
- Ordered through dealer
- Installed at dealership or NAPCO depot
- May or may not appear on build sheet
- Premium cost (~$750 over base)

## Production Information

### Production Timeline

| Period | Notes |
|--------|-------|
| **Late 1954** | 1955 First Series production begins |
| **Early 1955** | Continued production |
| **March 1955** | Second Series (Task Force) introduced |
| **March 1955** | First Series production ends |

### Production Numbers

Exact 1955 First Series production is difficult to determine as GM combined First and Second Series figures. Estimates:

| Series | Estimated First Series |
|--------|------------------------|
| **3100** | ~50,000 - 75,000 |
| **3600** | ~8,000 - 12,000 |
| **3800** | ~4,000 - 6,000 |
| **Total** | ~60,000 - 90,000 |

### NAPCO Production (1955 First Series)

Very limited data exists, but estimates suggest:
- **Very few** 1955 First Series trucks received NAPCO
- Most 1955 NAPCO installations were on Second Series
- First Series NAPCO trucks are exceptionally rare
- Perhaps only hundreds, not thousands, exist
