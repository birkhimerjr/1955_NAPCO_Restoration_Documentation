## VIN Decoding

### VIN Location
- Stamped on plate attached to **left door pillar** (hinge side)

### 1955 VIN Format

```
Example: H55A-001234

H     = Series (see chart below)
55    = Model Year (1955)
A     = Assembly Plant
001234 = Sequential Production Number
```

## Serial Number & Build Date Information

### Body Number Plate Location

- Attached to **left door pillar** below VIN plate
- Contains body style, trim, and paint codes

### Engine Number Location

- **235 Six:** Right side of block, rear
- **265 V8:** Front of block, below right head

### Fisher Body Date Coding

Stamped on body plate:
- First digit: Month (1-12)
- Second digit: Week (A-E)
- Third-Fourth digit: Year (55)
