## Axle Overview

### 1955 Chevrolet Truck Rear Axle Design

The 1955 Chevrolet trucks used a **Salisbury-type** rear axle with a removable differential carrier (often called a "pumpkin" or "third member"). This design allows the differential to be serviced without removing the axle from the vehicle.

```
                    REAR AXLE ASSEMBLY
    ┌─────────────────────────────────────────────────┐
    │                                                 │
    │   ┌─────┐                           ┌─────┐    │
    │   │BRAKE│                           │BRAKE│    │
    │   │DRUM │                           │DRUM │    │
    │   └──┬──┘                           └──┬──┘    │
    │      │                                 │       │
    │   ┌──┴──┐     ┌─────────────┐     ┌──┴──┐    │
    │   │WHEEL│     │ DIFFERENTIAL│     │WHEEL│    │
    │   │ HUB │◄────│   CARRIER   │────►│ HUB │    │
    │   └─────┘     │  (Pumpkin)  │     └─────┘    │
    │               └──────┬──────┘                 │
    │                      │                        │
    │              ┌───────┴───────┐                │
    │              │    PINION     │                │
    │              │    FLANGE     │                │
    │              └───────────────┘                │
    │                      │                        │
    │              To Driveshaft                    │
    └─────────────────────────────────────────────────┘
```

### Key Design Features

| Feature | Description |
|---------|-------------|
| **Housing Type** | Banjo-style (one-piece with removable carrier) |
| **Carrier Access** | Front-loading through large center opening |
| **Axle Retention** | C-clip (semi-floating) or bearing retained (full-floating) |
| **Brake Mounting** | Backing plates bolt to axle tube flanges |
| **Spring Perches** | Welded to axle tubes |

## Axle Identification

### Axle Code Location

The rear axle can be identified by:

1. **Axle Code Tag** - Riveted to differential cover or tube
2. **Casting Numbers** - On differential carrier and housing
3. **Date Codes** - Cast into housing

### Axle Code Tag Information

```
┌────────────────────────────┐
│   CHEVROLET MOTOR DIV.     │
│                            │
│   AXLE RATIO: 3.90         │
│   PLANT: XX                │
│   DATE: XX-XX              │
│                            │
└────────────────────────────┘
```

### Casting Number Locations

| Component | Location |
|-----------|----------|
| **Housing Center** | Top of banjo section |
| **Carrier** | Rear face of carrier casting |
| **Ring Gear** | Stamped on ring gear face |
| **Pinion** | Stamped on pinion head |

### Date Code Interpretation

Format: `A 15 5` or similar
- **First character:** Month (A=Jan, B=Feb, etc.)
- **Middle numbers:** Day of month
- **Last digit:** Year (5=1955)

### 3100 Series (1/2 Ton) - Semi-Floating

The 1/2 ton trucks used a **semi-floating** rear axle design.

| Specification | Value |
|---------------|-------|
| **Type** | Semi-floating |
| **Ring Gear Diameter** | 8.5" (approximate) |
| **Axle Shaft Spline** | 10 spline |
| **Bearing Retention** | C-clip at differential |
| **Brake Size** | 11" x 2" drums |
| **Bolt Pattern** | 6 x 5.5" (6 lug) |

**Semi-Floating Design:**
```
    WHEEL
      │
      ▼
┌───────────┐
│   HUB     │◄── Wheel studs press into hub
│  BEARING  │◄── Single bearing supports axle & wheel
│   AXLE    │◄── Axle shaft carries vehicle weight
│  SHAFT    │
└─────┬─────┘
      │
      ▼
  C-CLIP ◄── Retains axle in housing
      │
  DIFFERENTIAL
```

### 3600 Series (3/4 Ton) - Full-Floating

The 3/4 ton trucks used a **full-floating** rear axle design for increased load capacity.

| Specification | Value |
|---------------|-------|
| **Type** | Full-floating |
| **Ring Gear Diameter** | 9.5" (approximate) |
| **Axle Shaft Spline** | 10 spline |
| **Bearing Retention** | External spindle nut |
| **Brake Size** | 12" x 2" drums |
| **Bolt Pattern** | 8 x 6.5" (8 lug) |

**Full-Floating Design:**
```
    WHEEL
      │
      ▼
┌───────────┐
│   HUB     │◄── Hub rotates on spindle
│ BEARINGS  │◄── Two tapered bearings in hub
│ (2 sets)  │
│   AXLE    │◄── Axle only transmits torque
│  FLANGE   │◄── Bolts to hub
└─────┬─────┘
      │
  SPINDLE ◄── Supports all vehicle weight
      │
  AXLE HOUSING
```

## Gear Ratios

### Available Factory Ratios

| Ratio | Ring/Pinion Teeth | Primary Use | Notes |
|-------|-------------------|-------------|-------|
| **3.55:1** | 39/11 | Highway | Rare in trucks |
| **3.73:1** | 41/11 | Highway/light duty | Good fuel economy |
| **3.90:1** | 39/10 | Standard 3100 | Most common |
| **4.11:1** | 37/9 | Mixed use | Good all-around |
| **4.57:1** | 32/7 | Standard 3600 | Heavy loads |
| **4.88:1** | 39/8 | Off-road/heavy | With NAPCO |
| **5.14:1** | 36/7 | Severe service | Rare |

### Ratio Selection Guide

| Application | Recommended Ratio | Reasoning |
|-------------|-------------------|-----------|
| Highway cruising | 3.73:1 - 3.90:1 | Lower RPM, better fuel economy |
| Mixed use | 4.11:1 | Balance of power and economy |
| Towing/hauling | 4.57:1 - 4.88:1 | More pulling power |
| Off-road (NAPCO) | 4.57:1 - 4.88:1 | Matches front axle, better crawl |
| Larger tires | Go numerically higher | Compensate for effective ratio change |

### NAPCO Considerations

**Critical:** Front and rear axle ratios MUST match on NAPCO-equipped trucks.

| Front Axle Ratio | Required Rear Ratio |
|------------------|---------------------|
| 3.73:1 | 3.73:1 |
| 4.10:1 | 4.10:1 |
| 4.56:1 | 4.56:1 |
| 4.88:1 | 4.88:1 |

### Calculating Overall Ratio

```
Final Drive Ratio = Transmission Ratio × Transfer Case Ratio × Axle Ratio

Example: 1st gear crawl with 3-speed, NAPCO low, 4.11 axle:
= 2.94 × 1.94 × 4.11 = 23.45:1 crawl ratio
```

### Speedometer Calibration

Changing gear ratios affects speedometer accuracy:

| Original Ratio | New Ratio | Speedometer Error |
|----------------|-----------|-------------------|
| 3.90:1 | 4.11:1 | Reads ~5% slow |
| 3.90:1 | 4.56:1 | Reads ~17% slow |
| 3.90:1 | 3.73:1 | Reads ~4% fast |

*Speedometer gear changes may be required when changing ratios.*

### Gear Pattern Specifications

Proper gear tooth contact pattern is essential for quiet operation and longevity.

```
RING GEAR TOOTH (viewed from rear)

        ┌─────────────────────┐
        │                     │
   Heel │   ██████████████    │ Toe
        │   ██████████████    │
        │   ██████████████    │ ◄── Ideal: Centered,
        │   ██████████████    │     slightly toward toe
        │                     │
        └─────────────────────┘
              ▲
              │
           Root
```

**Pattern Adjustments:**

| Pattern Location | Problem | Adjustment |
|------------------|---------|------------|
| Too close to heel | Contact high on coast | Move pinion deeper (add shim) |
| Too close to toe | Contact high on drive | Move pinion shallower (remove shim) |
| Too high (crown) | Backlash too loose | Reduce backlash |
| Too low (root) | Backlash too tight | Increase backlash |

### Seal Installation Tips

1. **Clean sealing surfaces** thoroughly
2. **Apply thin film of RTV** to outer diameter of seal
3. **Coat seal lip** with gear oil or assembly lube
4. **Use proper driver** sized to seal outer race
5. **Drive seal squarely** to avoid cocking
6. **Verify seal depth** matches original

## Inspection & Service

### Pre-Disassembly Checks

1. **Check for leaks** - Note location and severity
2. **Listen for noise** - Drive, coast, turning
3. **Check backlash** - Through fill plug with dial indicator
4. **Inspect fluid** - Color, metal particles, smell
5. **Check axle play** - Grab tire, check for movement

### Disassembly Procedure Overview

**Semi-Floating (3100):**
1. Drain fluid
2. Remove wheels and drums
3. Remove carrier cover
4. Remove C-clips from axle shafts
5. Slide axle shafts out
6. Remove bearing retainer plates
7. Remove carrier assembly

**Full-Floating (3600):**
1. Drain fluid
2. Remove axle flange bolts
3. Slide axle shafts out
4. Remove hubs (spindle nut, bearings)
5. Remove carrier cover
6. Remove carrier assembly

### Measurement Points

| Measurement | Tool | Specification |
|-------------|------|---------------|
| Ring gear backlash | Dial indicator | 0.005" - 0.009" |
| Pinion depth | Pinion depth gauge | Per gear set specs |
| Carrier bearing preload | Dial indicator | 0.005" crush |
| Pinion bearing preload | Inch-pound torque wrench | 15-25 in-lbs (new) |
| Axle shaft runout | V-blocks and dial indicator | Max 0.010" |

### Reassembly Checklist

- [ ] Clean all components thoroughly
- [ ] Inspect all bearings and races
- [ ] Set pinion depth per gear manufacturer specs
- [ ] Install pinion with new crush sleeve
- [ ] Set pinion bearing preload
- [ ] Install carrier with proper shim pack
- [ ] Set ring gear backlash
- [ ] Check gear tooth pattern
- [ ] Install axle shafts with new seals
- [ ] Refill with proper fluid
- [ ] Test drive and re-check for leaks/noise

## Common Problems & Solutions

### Noise Diagnosis

| Noise Type | When it Occurs | Likely Cause | Solution |
|------------|----------------|--------------|----------|
| Whine | Acceleration only | Ring/pinion pattern | Adjust pinion depth |
| Whine | Deceleration only | Ring/pinion pattern | Adjust pinion depth |
| Whine | Both directions | Bearing wear | Replace bearings |
| Howl | Steady speed | Worn gears | Replace ring/pinion |
| Rumble | All conditions | Carrier bearing | Replace carrier bearings |
| Clunk | Direction change | Worn spider gears | Rebuild differential |
| Click | Constant rhythm | Chipped tooth | Replace ring/pinion |
| Growl | Cornering | Side bearing or axle bearing | Inspect and replace |

### Leak Diagnosis

| Leak Location | Cause | Solution |
|---------------|-------|----------|
| Pinion seal area | Worn seal or yoke | Replace seal, check yoke |
| Axle tube ends | Axle seal failure | Replace axle seals |
| Cover | Gasket failure | Replace gasket, check bolts |
| Housing cracks | Casting failure | Weld repair or replace |
| Vent | Overfilled or plugged vent | Check level, clear vent |

### Other Common Issues

| Problem | Cause | Solution |
|---------|-------|----------|
| Excessive backlash | Worn gears or bearings | Adjust or replace |
| No power to wheels | Broken axle shaft | Replace shaft |
| | Stripped splines | Replace shaft and side gear |
| Overheating | Low fluid | Check and fill |
| | Wrong fluid type | Drain and refill correctly |
| | Excessive preload | Re-adjust bearings |
| Vibration | Bent axle shaft | Replace shaft |
| | Out-of-round drum | Machine or replace drum |
| | Loose wheel bearing | Adjust or replace |

## Upgrades & Improvements

### Limited Slip Differential

Adding a limited slip (posi-traction) differential improves traction, especially valuable for NAPCO trucks.

| Option | Type | Notes |
|--------|------|-------|
| **Eaton Posi** | Clutch-type | Requires additive |
| **Detroit Truetrac** | Helical gear | No additive needed |
| **Auburn Gear** | Cone-type | Good for street |
| **Powertrax Lock-Right** | Locking | Full lock, some noise |

### Axle Shaft Upgrades

| Upgrade | Benefit |
|---------|---------|
| **Chromoly shafts** | Increased strength for off-road |
| **30-spline conversion** | Stronger than 10-spline |
| **Full-floating conversion** | Improved reliability |

### Gear Ratio Change

Consider ratio change for:
- Larger tires (go numerically higher)
- Highway driving (go numerically lower)
- Towing/off-road (go numerically higher)

**Remember:** NAPCO trucks must match front and rear ratios.

### Disc Brake Conversion

Aftermarket kits available to convert drum brakes to disc:
- Improved stopping power
- Better heat dissipation
- Easier maintenance
- May require different wheels

### Housing Upgrades

| Upgrade | Application |
|---------|-------------|
| **Truss kit** | Strengthens housing for off-road |
| **Differential cover** | Improved cooling, protection |
| **Axle tubes** | Larger diameter for strength |
